PROYECTO EDA2 - CARPETA: CABECERAS
----------------------------------
¿Que contiene esta carpeta?

    -> Esta carpeta contiene los archivos de cabecera (.h). Su función es definir el "QUÉ" hace el programa. 
    Acá se declaran las estructuras de datos, las clases y las firmas de las funciones para que el compilador 
    sepa que existen antes de que se ejecute la lógica.

ARCHIVOS CONTENIDOS:
- Entidades.h: Define el modelo de datos base (struct) para 'Ciudad'.
- Grafo.h: Define la clase GrafoLogistico sobre la que trabajará el programa.
- Persistencia.h: Declara las funciones necesarias para la lectura y escritura de archivos binarios .dat.
- Interfaz.h: Define la lógica de dibujo y los componentes visuales de la aplicación.


Tener en cuenta: Si se toca algo acá, probablemente haya que corregir cosas en los .cpp también. 
Tratemos de dejar las estructuras firmes.
